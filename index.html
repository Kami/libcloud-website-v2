One Interface To Rule Them All

Apache Libcloud is Python library that abstracts away differences among multiple cloud provider APIs.

It supports more than 30 providers such as:

and many others...

Features

  • Avoid vendor lock-in
  • Use the same API to talk to different providers
  • More than 30 supported providers
  • Four main APIs: Compute, Storage, Load Balancers, DNS

Installation

pip install apache-libcloud

Or download and install it manually from our servers.
